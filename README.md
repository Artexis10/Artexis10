# Hugo Ander Kivi

Founder of [Substrate Systems OÜ](https://substratesystems.io).

I build local-first infrastructure for people who want more control over their machines, memory, and work: external memory for agents, recoverable desktop state, and knowledge systems that keep source material attached to conclusions.

## Current Work

| Project | What it is | Shape |
| --- | --- | --- |
| [Exomem](https://github.com/Artexis10/exomem) | External memory for Codex, Claude Code, and other MCP clients over your own Markdown/Obsidian vault. Hybrid search, OCR/ASR/CLIP, and review queues for stale or conflicting notes. | Python, MCP, PyPI |
| [Endstate](https://github.com/Artexis10/endstate) | Declarative Windows machine backup and restore. Captures installed apps and settings so a fresh machine can be rebuilt intentionally. | Go, Tauri, Apache-2.0 |
| [Q](https://useq.ai) | AI knowledge infrastructure for creators and teams: turn source libraries into cited Q&A systems. | Multi-tenant, self-hostable |
