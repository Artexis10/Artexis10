## Hugo Ander Kivi

Founder of [Substrate Systems OÜ](https://substratesystems.io). Building tools that give people more control over their machines and their work.

### Active projects

**[Endstate](https://github.com/Artexis10/endstate)** — Declarative Windows machine provisioning. Captures installed apps and settings, restores them on a fresh install. Go + Tauri. Apache 2.0.

**[Q](https://useq.ai)** — AI-powered knowledge infrastructure for creators. Turn a YouTube library, docs, and PDFs into a branded AI Q&A system with citations back to the source. Multi-tenant, self-hostable.

**[kb-mcp](https://github.com/Artexis10/kb-mcp)** — A Model Context Protocol server that turns the Obsidian vault you already own into a multimodal knowledge base for Claude. Hybrid semantic + keyword search, with server-side OCR, transcription, and CLIP so images, audio, and video are findable by content — plus an audit layer that surfaces conflicting or stale notes. Python. AGPL-3.0.
